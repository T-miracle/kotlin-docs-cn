[//]: # (title: 协程)

异步或非阻塞编程是开发中的重要组成部分。
在创建服务器端、桌面 或 移动应用程序时，确保应用在用户体验上流畅且在需要时具有良好的扩展性至关重要。

Kotlin 通过在语言级别支持 [协程](https://en.wikipedia.org/wiki/Coroutine)，并将大部分功能委托给库，灵活地解决了这个问题。

除了开启异步编程的大门，协程还提供了许多其他可能性，如 并发模型 和 Actor 模型。
