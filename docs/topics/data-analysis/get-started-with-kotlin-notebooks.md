[Kotlin Notebook](kotlin-notebook-overview.md) 是一个交互式工具，允许您在单个文档中混合代码、可视化内容和 Markdown 文本。
您可以使用交互式编辑器在称为 _代码单元_ 的区块中编写和执行代码，即时查看结果并记录您的想法。
这种设置使其成为快速原型开发、分析和数据科学的理想工具。
